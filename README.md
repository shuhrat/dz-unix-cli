#SHIROQ

##Installation guide

* Clone project:
  `git clone https://github.com/Shuhrat/shiroq.git`

* Make a bin directory:
  `mkdir bin`

* Copy shiroq.sh to "bin":
  `cp shiroq.sh bin`
  
* Make shiroq.sh executable:
  `chmod u+x shiroq.sh`

* Add path to your `~/.bashrc` file
